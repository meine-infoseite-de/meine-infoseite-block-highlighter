Helper plugin for highlighting blocks within the block editor.

== Description ==

This plugin draws a subtle outline around block editor blocks (first four layers) and separates the title from the content.

*Be aware:* Once you worked with this plugin, you don't want to use the block editor without it anymore :)

Contributions are welcome via [Github](https://github.com/meine-infoseite-de/meine-infoseite-block-highlighter).

== Upgrade Notice ==

Upgrades just work.

== Frequently Asked Questions ==

*Are there any settings?*

No, the plugin has no settings.
